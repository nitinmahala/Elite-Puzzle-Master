Transform any image into a sliding puzzle game with multiple difficulty levels.

## 🌟 Features  
- 📸 Upload any image  
- 🧩 2×2 to 10×10 difficulty  
- ⏱ Timer & move counter  
- 👁 Original image preview  
- 🎉 Victory animations  
- 📱 Mobile responsive  
- 🌙 Dark/light mode  

## 🚀 Live Demo

- React 18  
- Tailwind CSS  
- Framer Motion
